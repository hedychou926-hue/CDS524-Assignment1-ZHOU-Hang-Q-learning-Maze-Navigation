This project presents a reinforcement learning project based on Q-learning for solving a maze navigation problem.
The goal of the agent is to move from a start cell to a goal cell while avoiding walls and traps and maximizing cumulative reward.

The notebook includes:

environment design,
random maze generation,
Q-learning implementation,
training and evaluation,
visualization of learning curves,
and path visualization of the learned policy.
This notebook is designed as part of the coursework for CDS524 Machine Learning for Business.
